Redis stores for ActionPack

redis-actionpack provides a session store for ActionPack, specifically for ActionDispatch. See the main redis-store readme for general guidelines.

Installation

# Gemfile
gem 'redis-actionpack'

Usage

If you are using redis-store with Rails, consider using the redis-rails gem instead. For standalone usage:

ActionController::Base.cache_store = ActionDispatch::Session::RedisSessionStore.new

Running tests

gem install bundler
git clone git://github.com/redis-store/redis-actionpack.git
cd redis-actionpack
bundle install
bundle exec rake

If you are on Snow Leopard you have to run env ARCHFLAGS="-arch x86_64" bundle exec rake
